## About

Eunyoung.dev는 Unity WebGL 기반으로 제작한
레트로 스타일 인터랙티브 포트폴리오입니다.

정적인 문서나 웹 페이지 대신,
작은 픽셀 공간을 탐험하고
오브젝트와 상호작용하며
게임처럼 포트폴리오를 경험할 수 있도록 구성했습니다.

---

## Why This Project

이 프로젝트는 AI 협업 워크플로우와
Harness 기반 개발 방식을 실험하는 과정에서 시작되었습니다.

짧은 기간 안에 완성 가능한 범위를 유지하면서도,
단순한 기능 데모보다는
하나의 작은 경험처럼 느껴지는 프로젝트를 만들고 싶었습니다.
